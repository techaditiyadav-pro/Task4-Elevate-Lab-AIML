# Logistic Regression Classifier - Breast Cancer Detection

This repository contains a solution for **Task 4: Classification with Logistic Regression**. It implements a binary classifier using the Breast Cancer Wisconsin Dataset to predict whether a tumor is Malignant or Benign.

## Project Structure
- `logistic_regression.py`: The main script containing data loading, preprocessing, modeling, and visualization.
- `requirements.txt`: List of dependencies required to run the project.
